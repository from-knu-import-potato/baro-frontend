### 시작하기 (Getting Started)

#### 1. 프로젝트 복제 및 의존성 설치

##### 1-1. 레포지토리 클론

```
git clone https://github.com/from-knu-import-potato/baro-frontend.git
```

##### 1-2. 프로젝트 폴더로 이동

```
cd baro-frontend
```

##### 1-3. 의존성 설치 (pnpm이 설치되어 있어야 합니다)

```
pnpm install
```

##### 1-4. Husky 세팅 활성화

```
pnpm run prepare
```

<br/>

#### 2. 개발 서버 실행

```
pnpm run dev
```
